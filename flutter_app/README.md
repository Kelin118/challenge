# Flutter Achievement Vault

Полноценная Flutter-версия приложения достижений в стиле Steam.

## Запуск

1. Установить Flutter SDK
2. Перейти в папку `flutter_app`
3. Если папки `android` и `ios` еще не созданы, выполнить `flutter create .`
4. Выполнить `flutter pub get`
5. Выполнить `flutter run`

## Что внутри

- темная Steam-style тема
- профиль игрока
- список достижений с поиском и фильтрами
- категории
- создание своих ачивок
- локальное хранение через `shared_preferences`
- экран деталей
- экран статистики
